# MH2 AI Chatbot

A modern AI chatbot web application with a sleek glassmorphism UI and a backend-powered data system that fetches real-time business and website information from external APIs.

## Features
- Real-time chatbot UI (HTML/CSS/JS)
- Backend API integration for live data
- Business & website information retrieval system
- External API-based data fetching (Google Places / Web APIs)
- Smart responses (time, date, general Q&A)
- Modern glassmorphism UI design
- Fully responsive interface
- Expandable AI logic system


## Tech Stack

| Frontend | Backend |
|----------|----------|
| HTML5 | App Script |
| CSS3 | Google Sheet |
| Vanilla JavaScript | |

## Data Sources:
- Google Places API (business info)
- Web Search APIs (SerpAPI / Bing API optional)
- Custom JSON database (fallback system)
